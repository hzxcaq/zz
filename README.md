# 极简 [![Build Status](https://www.travis-ci.org/libp/zz.svg?branch=master)](https://www.travis-ci.org/libp/zz)

## 介绍
* 基于springboot4.0+mybatis实现前后端分离
* 面向接口编程的极简项目
* 提供纯后端的API调用功能  
* 项目中包含数据库初始化sql

## 已实现接口

* 获得随机提供的文章
* 获得每日推荐文章
* 基于ID获取文章

## 安装使用
* gitclone到本地，IDEA导入
* 执行sql初始化命令
* 文章内容通过python爬虫取自互联网，可参考SpiderScrapyArticle工程
